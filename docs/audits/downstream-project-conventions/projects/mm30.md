## 2. File And Directory Organization

Static-concept families use slug-only (genes, topics, questions, datasets, papers, mechanisms, methods).
- **`docs/` vs `doc/`:** *stale or unclear convention.* Both exist. `docs/` holds only `superpowers/{plans,specs}/` (14 files) — older Superpowers-era plans/specs from before the `doc/plans/` and `doc/specs/` taxonomy stabilized.
